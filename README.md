## install for  linux
***
```
sudo apt update && apt upgrade

sudo apt install clang gcc -y

git clone https://github.com/RedMads/guess_game.git

cd guess_game

g++ main.cpp -o Guess_game

chmod +x Guess_game

./Guess_game
```
